# Adele-Voice-Pack-for-Frsky-Ethos
A voice pack for Frsky's Ethos operating system.  Contains full system set of audio and 165+ additional audio files which should cover most fixed wing, multirotor, FPV and flight controller setups.

I dont know much about helis or gliders so may be lots of phrases missing for those disciplines - please let me know.

More details to come.  Copy Adele folder to /audio/en

If you find incorrect callouts, audio issues or want phrases added, pls raise an issue.
